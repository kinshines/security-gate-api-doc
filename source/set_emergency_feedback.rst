============================
控制闸机紧急状态异步反馈
============================

场景描述
----------
服务端发送 `控制闸机紧急状态的指令 <set_emergency.html>`_ 后，闸机通过此接口异步反馈闸机的紧急状态。

面向客户端
::::::::::::::::::::
闸机

请求URL
---------------------
**HTTP POST**  /api/gatefeedback/emergency

请求体格式
----------------------------
请求体格式为JSON，字段名不区分大小写::

    {
        "deviceCode":"string", //闸机设备编号
        "sbe":"int", //报警状态, 1:紧急;0:正常
        "execute":"string" //执行结果说明
    }

请求体示例
----------------------------
::

    {
        "deviceCode":"Z001",
        "sbe":1,
        "execute":"OK"
    }
