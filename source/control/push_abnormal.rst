============================
闸机异常状态上报
============================

场景描述
----------
当闸机在正常使用过程中，人为异常闯入闸机，闸机通过此接口上报反馈闸机的异常状态。

面向客户端
::::::::::::::::::::
闸机

请求URL
---------------------
**HTTP POST**  /api/gatefeedback/abnormal

请求体格式
----------------------------
请求体格式为JSON，字段名不区分大小写::

    {
        "deviceCode":"string", //闸机设备编号
        "status":"int" //异常状态, 1:正向闯入; 2:正向闯入解除; 3:反向闯入; 4:反向闯入解除; 5:异常进入安检闸机; 6:异常进入安检闸机解除
    }

请求体示例
----------------------------
::

    {
        "deviceCode":"Z001",
        "status":1
    }
