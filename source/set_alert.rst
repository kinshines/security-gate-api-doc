====================
控制闸机报警
====================

场景描述
----------
通过此接口控制闸机报警，默认报警时间为20秒

请求URL
---------------------
**HTTP POST**  /api/gate/alert?sec={sec}
参数 **{sec}** 可设置报警时长（秒），若设置为0，则停止报警

请求体格式
----------------------------
请求体格式为JSON，字段名不区分大小写::

    {
        "Id":"long", //闸机设备ID,若值不为0,根据此字段查询设备,若为0或为提供,则根据DeviceCode查询设备
        "DeviceCode":"string" //闸机设备编号
    }

请求体示例
----------------------------
::

    {
        "Id":1
        "DeviceCode":"Z001"
    }
