====================
控制闸机报警
====================

场景描述
----------
通过此接口控制闸机报警，默认报警时间为20秒

请求URL
---------------------
**HTTP POST**  /api/gate/alert?sec={sec}

参数 **{sec}** 可设置报警时长（秒），若设置为0，则停止报警，默认报警时间为20秒

请求URL示例
----------------------------
**HTTP GET**  /api/gate/alert?sec=20

请求体格式
----------------------------
请求体格式为JSON，字段名不区分大小写::

    {
        "id":"long", //闸机设备ID,若值不为0,则根据此字段查询设备,若为0或为提供,则根据DeviceCode查询设备
        "deviceCode":"string" //闸机设备编号
    }

请求体示例
----------------------------
::

    {
        "id":1
        "deviceCode":"Z001"
    }
