==========================
紧急状态推送
==========================

场景描述
-------------
按动闸机紧急按钮，闸机软件进入紧急状态，并将紧急状态推送至安检信息系统。
取消紧急后，恢复到紧急前的状态，并将正常状态推送至安检信息系统。

面向客户端
::::::::::::::::::::
闸机

请求URL
-------------------------
**HTTP POST**  /api/gate/pushemergency

请求体格式
----------------------------
请求体格式为JSON，字段名不区分大小写::

    {
        "deviceCode":"string",//设备编号
        "emergency","int"//紧急状态,0:正常;1:紧急
    }

请求体示例
----------------------------
::

    {
        "deviceCode":"Z001",
        "emergency",1
    }
