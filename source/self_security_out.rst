==========================
自助安检闸机旅客外出请求
==========================

场景描述
----------
旅客进入安检区后，因某些原因要外出安检区，应在工作人员指引下，在闸机上反向扫描证件，闸机将扫描到的证件信息发送至安检信息系统，安检信息系统将记录旅客的外出状态，
需要注意的是，旅客扫描证件后，将由工作人员引领外出，而非通过安检闸机外出，因此闸机栅门无需作出响应

面向客户端
::::::::::::::::::::
闸机

请求URL
-------------------------
**HTTP POST**  /api/selfsecurity/out

请求体格式
----------------------------
请求体格式为JSON，字段名不区分大小写::

    {
        "deviceCode":"string",//设备编号
        "type":"string", //证件类型, NI:身份证; PP:护照; PPOCR:OCR模式识别的护照; TP:港澳通行证; TW:台湾通行证; HM:港澳居民来往内地通行证; TB:台湾居民来往大陆通行证; HMT:港澳台居民居住证; FR:外国人永久居留身份证; ID:其他
        "id":"string"//证件号
    }

请求体示例
----------------------------
::

    {
        "deviceCode":"Z001",
        "type":"NI",
        "id":"110101197608056896"
    }
