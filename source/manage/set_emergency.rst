====================
控制闸机进入紧急状态
====================

场景描述
----------
通过此接口控制闸机进入紧急状态，
注意此接口仅发送指令，闸机通过 `闸机紧急状态反馈接口 <../control/set_emergency_feedback.html>`_ 异步返回闸机的紧急状态。

面向客户端
::::::::::::::::::::
安检管理后台和移动管理平台

请求URL
---------------------
**HTTP POST**  /api/gate/emergency

请求体格式
----------------------------
请求体格式为JSON，字段名不区分大小写::

    {
        "deviceCode":"string", //闸机设备编号
        "emergency":"int" //闸机紧急状态,1:设置紧急状态;0:取消紧急状态
    }

请求体示例
----------------------------
::

    {
        "deviceCode":"Z001",
        "emergency":1
    }
